Preamplifier Power

The heart of any interface is its ability to cleanly amplify a microphone signal. We compare three key metrics: Gain Range, Dynamic Range, and self-noise (EIN).

Mic Preamp Gain Range (dB)

More gain allows you to use quieter microphones without needing an external booster. Higher is better.

Input Dynamic Range (dB)

The difference between the loudest and quietest sounds an interface can capture. A wider range means cleaner, less noisy recordings. Higher is better.

Equivalent Input Noise (EIN in dBu)

A measure of the preamp's self-noise. A more negative number indicates a quieter preamp. (Note: Steinberg does not specify this metric).
